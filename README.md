# 画像検索アプリ

## 紹介と使い方

- pixabay の WebAPI を利用して画像検索アプリを作成した

- 検索バーに文字を入力して Enter を押すと、画像を検索することができる

- 画像をクリックすると別タブが開く画像保存できるページに遷移する

## 工夫した点

- useState と useRef を使用して作成
